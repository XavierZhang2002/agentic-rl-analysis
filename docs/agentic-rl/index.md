## 报告结构

### [第一篇：算法与关键技术](./01-算法与关键技术.md)

从推理 RL 到 Agentic RL 的本质变化，以及有影响力的算法介绍：

- **一、从推理 RL 到 Agentic RL** — 核心矛盾与四大挑战（奖励信号 68%、训练稳定性 53%、探索效率 44%、信用分配 44%）
- **二、奖励信号** — IGPO（信息增益内在奖励）、CM2（多维度 Checklist）及其他方案
- **三、训练稳定性** — SeeUPO（首个多轮收敛保证）、ARLArena/SAMPO（系统分析）、VCPO（方差控制）
- **四、探索效率** — EMPO²（记忆增强探索 +128.6%）、LUFFY（Off-Policy 混合）
- **五、信用分配** — GiGPO（锚点状态）、ELPO（错误定位）、ProxMO（语义邻近性）
- **六、Agentic 工程实践** — GLM-5（异步基础设施）、Kimi K2（3000 工具）、环境构建
- **七、算法速查与对比** — 全部算法一览、影响力分级、技术路线图

### [第二篇：综合分析与展望](./02-综合分析与展望.md)

主观评价、分析与未来展望：

- **一、领域全景** — 2024-2026 时间线、论文与机构分布
- **二、技术路线分析** — 三条路线（修复基础算法 / 重新建模 / 增强基础设施）的比较
- **三、产业观察** — 已落地产品、开源生态、中美对比
- **四、核心判断** — 6 条个人观点（💭 标记）：信用分配本质、SeeUPO 实践意义、记忆机制前景...
- **五、未来方向** — 短期/中期/长期预测
